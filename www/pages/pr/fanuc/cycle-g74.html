Циклы сверления G74
G74 - Цикл сверления торцевой поверхности с периодическим выводом сверла

Этот цикл позволяет стружкодробление при обработке внешнего диаметра. Если вторая ось на плоскости (ось X (ось U) для плоскости ZX) и адрес P не указаны, то операция выполняется только вдоль первой оси на плоскости (ось Z для плоскости ZX), то есть, выполняется цикл сверления с периодическим выводом сверла.

Программирование
G74 R...
G74 X... Z... P... Q... R... F...
Параметры
Первая строка

R (e) - Величина возврата. Это значение является модальным и не изменяется, пока не будет задано другое значение.

Вторая строка

X - Координата оси X на плоскости (ось X для плоскости ZX) в точке B.

Z - Координата оси Z на плоскости (ось Z для плоскости ZX) в точке C.

P (∆i) - Расстояние перемещения в направлении второй оси на плоскости (ось X для плоскости ZX).

Q (∆k) - Глубина резания в направлении первой оси на плоскости (ось Z для плоскости ZX).

R (∆d) - Величина отвода инструмента при проходе на конечную точку.

F - Скорость подачи.

Схема обработки

Траектория резания в цикле сверления в торце с периодическим выводом сверла

Пояснение

Повторяется циклическая операция резания по ∆k и возврата по e. При проходе до точки C инструмент отводится на ∆d. Затем инструмент возвращается на скорости ускоренного перемещения, перемещается в направлении точки B на ∆i и снова выполняет резание.
